# Image Asset Sources — WNB

Dokumen ini mendokumentasikan seluruh sumber gambar stock yang digunakan secara sementara (*temporary stock illustration*) pada aplikasi **WNB (Wisata Ngaprak Bandung)**.

## Kebijakan Penggunaan Aset Visual
1.  **Status Aset:** Semua foto stock yang digunakan di halaman utama berstatus sebagai **ilustrasi visual sementara** (*temporary stock illustration*) dan akan diganti dengan aset foto resmi WNB jika sudah dikonfirmasi.
2.  **Kejujuran Visual:** Aset foto yang memperlihatkan unit kendaraan wajib menampilkan catatan keterang kecil (*notice*): `"Foto ilustrasi — bukan dokumentasi armada WNB"` demi menghindari misrepresentasi armada komersial WNB.
3.  **Metode Penyimpanan:** Semua aset diunduh secara penuh dan disimpan langsung secara lokal pada direktori repository `client/public/images/home/` dalam format WebP yang dioptimalkan untuk performa pemuatan cepat. Penggunaan tautan luar (*hotlink*) dilarang keras.
4.  **Lisensi:** Seluruh gambar bersumber dari platform **Pexels** dan berlisensi bebas pakai untuk komersial maupun nonkomersial di bawah ketentuan [Pexels License](https://www.pexels.com/license/).
