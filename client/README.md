# WNB Client (Frontend)

Folder ini berisi kode sumber untuk frontend aplikasi **Wisata Ngaprak Bandung (WNB)**.

## Rencana Teknologi
- **UI Framework:** React
- **Bahasa:** TypeScript
- **Build Tool:** Vite
- **Styling:** Tailwind CSS

> [!NOTE]
> Teknologi di atas **belum diinisialisasi** pada tahap ini.
> Seluruh implementasi dan konfigurasi frontend akan dilakukan pada berkas rencana implementasi (*implementation plan*) berikutnya.
